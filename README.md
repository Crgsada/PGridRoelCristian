# Práctica CSS Grid

## Autor
Cristian Roel Gómez

## Descripción
Página web realizada como práctica de CSS Grid para la asignatura de Lenguaje de Marcas.

El ejercicio requería el uso de CSS Nesting, pero heoptado por usar selectores de clase descendentes. El motivo es que el validador de CSS (W3C) detectaba errores. Las lineas afectadas son:
*/LINEA 154-185*/
.fila7 h2,
.fila7 p,
.fila7 ul {
    margin: 0;
}
*/Linea 72-74*/
figure {
    margin: 0;
}

## Tecnologías utilizadas
- HTML5
- CSS3
- CSS Grid
- Flexbox
